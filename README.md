# Features
- Handle Creation Callback ( Process & Threads )
- Heartbeat System
- Overlay Scanner
- Driver Scanner ( WIP )
- Anti Debug
- Anti Kill / Suspend Threads
- Anti Injection
- Digital Signature Scanner
- Anti TestMode
- Signature Scanner
